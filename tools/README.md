# tools/

Инструменты сопровождения вики. Обычному читателю сайта они не нужны — это
рабочая оснастка для тех, кто вики пополняет.

## chat.py — поиск по экспорту чата

Вики строится из экспорта истории Telegram-группы. Каждое утверждение
сопровождается ссылкой на сообщение-источник, и эти ссылки надо откуда-то брать:
формат `t.me/aimairn/<topic_id>/<msg_id>`, но поля «топик» в экспорте Telegram
нет — оно вычисляется по цепочке ответов.

```bash
tools/chat.py 'deferimento t[áa]cito'                # поиск по всему чату
tools/chat.py 'tácito' --topic 43113 --min-len 800   # ветка + длинные сообщения
tools/chat.py '#кейспортал' --limit 50 --chars 400
tools/chat.py --id 110356                            # прочитать сообщение целиком
tools/chat.py --help                                 # в т.ч. список веток
```

Полезные фильтры при раскопках: `--min-len 600` (развёрнутые гайды обычно
длинные), `--topic` (ветка), поиск по хештегам `#кейспортал`, `#статистика`,
`#кейсaima` — это структурированные кейсы-таймлайны.

Модуль можно импортировать: `load()`, `search()`, `root_of()`, `link()`.

## check_wiki.py — проверки перед публикацией

```bash
tools/check_wiki.py
```

1. **Описания страниц.** У каждой страницы должен быть `description:` во front
   matter — иначе в превью ссылки (Telegram, Slack, WhatsApp) подставится общее
   описание сайта, одинаковое для всех страниц.
2. **Ссылки на сообщения.** Сообщение существует в экспорте, и ветка в URL
   совпадает с вычисленной. Иначе читатель проваливается не туда — а
   проверяемость источников здесь основная ценность.

Вызывается из `release.sh`, так что при публикации через него отдельно запускать
не нужно.

## Данные

Оба скрипта работают с каталогом `Чат продления ВНЖ/` — он **не хранится в
репозитории** (большой, содержит персональные данные). Без него `chat.py` не
запустится, а `check_wiki.py` пропустит проверку ссылок и проверит только
описания.

Первый запуск строит кэш в `.cache/` (нужен `jq`, `sudo apt install jq`);
кэш пересобирается автоматически, если экспорт обновился.
