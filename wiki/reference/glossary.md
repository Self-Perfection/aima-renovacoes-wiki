# Глоссарий терминов

Португальские и специфические термины, встречающиеся при продлении ВНЖ.

## Портал и статусы заявки

- **AIMA** — Agência para a Integração, Migrações e Asilo. Ведомство по миграции,
  преемник SEF.
- **SEF** — Serviço de Estrangeiros e Fronteiras. Прежнее миграционное ведомство.
- **Título de Residência (TR)** — карта ВНЖ.
- **Pedido** — заявка/запрос. **Pedido id** — номер заявки (см. [Этапы](../process/etapy-zayavki.md)).
- **Novo Pedido** — кнопка подачи новой заявки.
- **Submetido** — подана. **Aguarda Pagamento** — ожидает оплаты.
  **Carregamento de Documentos** — загрузка документов. **Avaliação** —
  рассмотрение. **Concluído / Deferido** — завершено/одобрено. **Indeferido** —
  отказано. Подробно — [Этапы заявки](../process/etapy-zayavki.md).
- **Recibo Comprovativo** — PDF-подтверждение подачи с QR-кодом.
- **Token** — публичная ссылка для проверки статуса без входа (из QR).
- **Audiência Prévia / Alegação** — дозапрос/возражение, см. [Дозапрос](../problems/dozapros.md).
- **Proposta de indeferimento** — предварительный отказ (можно возразить).
- **Deferimento tácito** — молчаливое согласие (авто-одобрение через 60 раб. дней),
  см. [Застревание](../problems/zastrevanie.md).

## Данные и номера

- **NIF** — налоговый номер (Número de Identificação Fiscal).
- **NISS** — номер соцстрахования (Segurança Social).
- **NIE / NIPS** — внутренний ID иностранца в системе AIMA.
- **Utente** — номер в системе здравоохранения SNS.
- **Morada** — адрес (в системе AIMA — печатается на карте).
- **Despacho** — код статьи ВНЖ, см. [Статьи ВНЖ](../process/stati-vnzh.md).
- **CMD** — Chave Móvel Digital, цифровая подпись/аутентификация.

## Документы

- **Modelo 1** — основное заявление на портале.
- **Modelo 4** — документ о предоставлении средств (для воссоединения),
  подпись заверяется у нотариуса.
- **Requerimento Inicial** — раздел для прочих документов.
- **Comprovativo** — подтверждение/справка.
- **Domicílio Fiscal** — налоговый адрес (из Finanças, как подтверждение адреса).
- **Certidão Permanente** — выписка о недвижимости (predialonline.pt).
- **Caderneta Predial** — налоговая карточка объекта недвижимости.
- **Registo Criminal** — справка о несудимости.
- **Certidão de deferimento tácito** — сертификат молчаливого согласия.

## Организации и сервисы

- **AT (Autoridade Tributária) / Finanças** — налоговая.
- **Segurança Social (SS)** — соцслужба.
- **SNS** — Serviço Nacional de Saúde, госмедицина.
- **CTT** — почта Португалии (доставка карт).
- **Junta (de Freguesia)** — местная администрация (справки об адресе).
- **IRN** — Instituto dos Registos e Notariado (раньше вело часть продлений).
- **Provedoria de Justiça** — омбудсман.
- **Livro Amarelo** — официальная жалобная книга в госучреждениях.
- **Loja AIMA** — отделение AIMA (личный приём).

## Статусы долгов (Situação AT/SS)

- **Em Revisão** — идёт проверка. **Regularizado** — долгов нет.
  **Isento** — освобождён. **Por Regularizar** — есть долг.
  См. [Задолженности](../prerequisites/zadolzhennosti.md).

## Сленг чата

- **«Послетридцатники»** — те, чей ВНЖ истекает после 30.06.2025 (этот портал).
- **«Дотридцатники»** — истекающие до 30.06.2025 (соседний топик/порядок).
- **«Хайквал» / «высококвал»** — высококвалифицированный специалист (G11/G14, 90.2).
- **«Референс»** — референс для оплаты (Referência), см. [Оплата](../process/oplata.md).
- **«Апрув»** — одобрение (Deferido).
- **«Дозвон»** — прозвон колл-центра AIMA / одноимённый чат.
