# Процесс продления

Пошаговый путь продления ВНЖ через портал AIMA Renovações — от входа на портал до
получения карты. Пайплайн статусов заявки:

```
Submetido → Pagamento → Documentos → Avaliação → Concluído (Deferido)
```
